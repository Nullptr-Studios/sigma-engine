\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Ch = char$>$\newline
class rapidxml\+::xml\+\_\+attribute$<$ Ch $>$}
Class representing attribute node of XML document. Each attribute has name and value strings, which are available through \doxylink{classrapidxml_1_1xml__base_a72e6615cd5b3133ab6c8df72c2fdcd20}{name()} and \doxylink{classrapidxml_1_1xml__base_af3a3d4534f4c88f151e0f063aef1c6de}{value()} functions (inherited from \doxylink{classrapidxml_1_1xml__base}{xml\+\_\+base}). Note that after parse, both name and value of attribute will point to interior of source text used for parsing. Thus, this text must persist in memory for the lifetime of attribute.
