\doxysubsection{Detailed Description}
Class that handles the process of loading and playing sounds by wrapping FMOD\textquotesingle{}s functionality. Deals with all FMOD calls so that FMOD-\/specific code does not need to be used outside this class. Only one \doxylink{class_f_n_f_e_1_1_audio_engine}{Audio\+Engine} should be constructed for an application.
